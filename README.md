# 🐧 What is UNICA? 💻
**Unica (Unix Command Based)** is a shell-based OS and it's **OPEN SOURCE!** 📂 You can edit anything on UNICA! 🛠️
Unica was created specifically for adding custom commands and changing your terminal UI to look more pro! 🚀
# 🛠️ What can I do with UNICA?
You can:  
1.⌨️ Use/Make a custom command on your terminal.  
2.🎨 Edit/Make a custom UI on your terminal.  
3.👤 Edit your username.  
4.🐧 Use default Linux commands.  
# 🚀 How to run UNICA?
## 🐧 Run on Linux:
You need:  
1.🖥️ Linux Terminal  
2.📄 Unica shell file  
Follow these steps:  
1.📥 Download the Unica shell file.  
2.💻 Open your terminal.  
3.📂 Go to the Unica directory: cd [path to unica shell file]  
4.⚡ Run it: sh [file-name].sh  
5.💥 BOOM! You have Unica running on your Linux!  
## 🤖 Run on Android:
You need:  
1.📱 Linux terminal (Use Termux or TermOne Plus)  
2.📄 Unica shell file  
Follow these steps:  
1.📥 Download the Unica shell file.  
2.🔓 Open Termux or TermOne.  
3.📂 Go to the Unica directory: cd [path to unica shell file]  
4.⚡ Run it: sh [file-name].sh  
5.💥 BOOM! You have Unica running on your Android!
