# What is UNICA?
Unica (Unix Command Based) Is a shell based OS and ITS OPEN SOURCE!. You can edit anything on UNICA!
Unica was created only for adding custom commands and change your terminal ui 
# What i can do with UNICA?
You can:
1.Use/Make a custom command on your terminal
2.Edit/Make a custom ui on your terminal
3.Edit username
4.Use default linux cmds
# How to run UNICA?

# Run on Linux:
You need:
1.Linux terminal
2.Unica zip image
3.Unzip (In some distro you need to install this first to unzip the image)
And now follow these steps:
1.Download the unica zip image
2.Open terminal
3.Type "sudo apt install unzip" (In CentOS/RHEL/Fedora you need to type "sudo yum install unzip" or "sudo dnf install unzip")
4.After install done. Type unzip "[zip-file-name].zip" 
5.Ok,After extract done. Now go to unica directory by typing the "cd [path to unicaos folder]" command
6.Now type "sh osmain.sh" to run unica 
7.BOOM You have unica running on your Linux
# Run on Windows:
You need:
1.Linux terminal (Dont use windows terminal!. you need gitbash on github app or ubuntu app on micro store)
2.Unica zip image
3.winrar
And now follow these steps:
1.Download the unica zip image
2.Extract the zip image
3.Open github app or ubuntu app
4.On github app open gitbash. On ubuntu app open terminal
5.Now go to unica directory by typing the "cd [path to unicaos folder]" command
6.Now type "sh osmain.sh" to run unica 
7.BOOM You have unica running on your Windows
